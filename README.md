# t_coussot_cleaner

## Installation

```
git clone https://github.com/Nephylhim/t_coussot_cleaner.git ~/catkin_ws/src/t_coussot_cleaner
```

## Usage

```
cd ~/catkin_ws
roscore
```
In another terminal :
```
rosrun turtlesim turtlesim_node
```
And in a third terminal :
```
rosrun t_coussot_cleaner t_coussot_cleaner
```
